# Pong Game

## Description
A simple two-player Pong game implemented using Pygame.

## Requirements
- Python 3.x
- Pygame library

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/Ish-xo/Pong_game.git
   cd Pong_game
   ```
2. Install Pygame:
   ```bash
   pip install pygame
   ```
3. Run the game:
   ```bash
   python pong.py
   ```

## Controls
- Player 1: W (up), S (down)
- Player 2: UP (up), DOWN (down)
